# Tool Icons

Place your PNG icon files here.

Recommended specs:
- Format: PNG with transparency
- Size: 64x64 or 128x128 pixels
- Naming: kebab-case (e.g. react.png, tailwind-css.png, vs-code.png)

Then import them in src/components/sections/Tools.jsx:

```js
import reactIcon from '../../assets/icons/react.png'
```

See README.md for full instructions.
