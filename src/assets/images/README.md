# Project Images

Place your project screenshot/preview images here.

Recommended specs:
- Format: WebP (preferred) or PNG/JPG
- Size: 1200x750 pixels (16:10 ratio)
- Naming: project1.webp, project2.webp, etc.

Then import in src/components/sections/Projects.jsx:

```js
import proj1 from '../../assets/images/project1.webp'
```

See README.md for full instructions.
